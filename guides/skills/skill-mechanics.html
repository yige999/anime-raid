📚 技能系统概述

⚔️ 主动技能

需要手动释放的技能，包含攻击、防御、辅助等多种类型。是角色战斗的核心。

  • • 消耗能量释放
  • • 有冷却时间
  • • 可手动升级
  • • 决定战斗策略

🛡️ 被动技能

自动生效的常驻技能，提供属性加成和特殊效果，增强角色综合实力。

  • • 自动触发
  • • 无需消耗
  • • 永久生效
  • • 基础实力保障

⚡ 连携技

多个角色配合释放的强力技能，需要特定角色组合和条件触发。

  • • 需要角色配合
  • • 超强威力
  • • 战术核心
  • • 团队协作体现

攻击型技能

单体攻击技能

针对单个敌人造成高额伤害，适合Boss战和PvP。

高伤害 精确打击 核心输出

同时攻击多个敌人，适合清怪和群体战斗。

施加持续伤害效果，长期战斗中有优势。

DoT 持续输出 战术选择

治疗 生存保障 团队核心

护盾技能

提供伤害吸收护盾，关键时刻保护队友。

🛡️ 被动技能系统

🔥

属性强化型

直接提升角色基础属性，包括攻击、防御、生命等。

  • • 攻击力 +15%
  • • 防御力 +10%
  • • 生命值 +20%
  • • 暴击率 +5%
⚡

效果触发型

在特定条件下触发，提供额外效果或加成。

  • • 生命低于30%时攻击+25%
  • • 暴击时恢复生命值
  • • 受击时减少伤害
  • • 释放技能后加速
💚

环境适应型

根据战斗环境或队伍配置提供适应性加成。

  • • 同属性队友+10%攻击
  • • 特定地图效果增强
  • • 人数优势时加成
  • • 时间流逝效果

⚡ 连携技系统

🔗 连携条件

1

角色组合

需要特定的角色组合同时在场，如火+水、光+暗等属性组合。

2

能量要求

参与连携的所有角色都需要有足够的能量（通常为50%以上）。

3

时机把握

在战斗特定时机或满足条件时才能触发，如Boss战阶段转换。

💫 连携效果

元素爆发连携

造成超大范围元素伤害，并施加相应元素效果。

治疗护盾连携

为全队提供大量治疗和强力护盾。

增益强化连携

大幅提升全队属性，持续较长时间。
